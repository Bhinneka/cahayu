
## Cahayu, a basic structure app using SparkJava http://sparkjava.com

## Getting started

- Install `Maven`
https://maven.apache.org/download.cgi

- Build `jar`
```shell
$ mvn clean package
```

- Run `jar`
```shell
$ java -jar target/Cahayu-1.0-SNAPSHOT.jar
```

## Run Tests

```shell
$ mvn test
```
